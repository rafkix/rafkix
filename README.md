## ⚡ Featured Projects

### 🔐 [Auth Service](https://github.com/rafkix/auth-service)
> **Production-ready authentication ecosystem built with modern security best practices.**
- Implements secure **JWT Access/Refresh token** cycles with rotation & proactive revocation.
- Integrated active device and session tracking mechanisms.
- Utilizes a high-performance **Redis-backed blacklist** for instant token invalidation.
- *Technologies:* `FastAPI` • `Redis` • `PostgreSQL`

### 📦 [API Template](https://github.com/rafkix/api-template)
> **A scalable, highly optimized backend boilerplate designed for rapid enterprise scaling.**
- Built strictly on **Clean Architecture** patterns (Decoupled Services & Repositories).
- Centralized global exception handler ensuring predictable, structured API responses.
- Production-configured layout setup for direct deployment.
- *Technologies:* `FastAPI` • `PostgreSQL` • `Docker`

### ⚡ [Rate Limiter](https://github.com/rafkix/rate-limiter)
> **Lightweight, distributed traffic control system built for modern APIs.**
- Engineered to safeguard applications against DDoS attacks and brute-force API abuse.
- Uses **Redis atomic operations** to guarantee sub-millisecond rate-limit checks.
- *Technologies:* `Python` • `Redis`
